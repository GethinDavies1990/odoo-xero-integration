Odoo Xero Connector

Odoo is an Enterprise Resource Planning(ERP) system which includes CRM, Inventory, Warehouse etc. Xero is a cloud-based accounting software platform for small and medium-sized businesses. Odoo integration with Xero make it easy to sync the data i.e Customers, Products, Invoices and Purchase Orders.



Features



Sync Customers in ODOO and Xero both ways

Sync Products in ODOO and Xero both ways

Import Invoices from Xero to ODOO

Import Purchase Orders from Xero to ODOO

Installation

  • Download ODOO Xero Connector module and place into addons directory
  • Restart Odoo server / service
  • Open setting, activate Developer Mode
  • Go to Apps
  • Update Apps List from top menu
  • Search the Xero and install it
  • Happy to use Xero


How Its Work


  • Go to Xero Developer Account
  • Click to New App button

  • Fill the application name field
  • Select Integration type i.e Web App
  • Fill the redirect url field i.e [domain.com]/xero_success
  • Click to Create App button

  • Collect credentials information (Client ID, Client Secret, Redirect URI)

  • Install ODOO Xero Connector Module
  • Go to Xero from menu
  • Fill credential values collected from Xero Developer App
  • Click on Connect button
  • It will open an authorization page
  • After authentication, it will redirect to Redirect-URI(ODOO page).

  • Select specific option(s)
  • Select Xero tenant
  • Adjust time limiter
  • Click on Sync button

  • Import History menu shows how many records are imported
  • There will be no record if nothing is imported.

  • Export History menu show how many records are exported
  • There will be no record if nothing is exported.
